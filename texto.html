Principais Formatações de Textos

Negrito

A Tag STRONG deixa o texto em NEGRITO

Itálico

A Tag "EM" deixa o texto em Itálico

Marca Texto

A Tag "MARK" deixa o texto marcado, ESSA PARTE DO TEXTO É IMPORTANTE

Riscando / Excluindo Texto

A Tag "DEL" risca um texto GATILHO

Sublinhado / Inserindo Texto

A Tag "INS" sublinha um texto AQUI VAI FICAR O SEU NOME

Texto Sobrescrito

A Tag "SUP" deixa o texto Sobrescrito Ex.: 2x2 + 5x - 3

Texto Subscrito

A Tag "SUB" deixa o texto Subscrito Ex.: CO2 / H2O
